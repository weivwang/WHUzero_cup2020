#  零杯网页设计比赛 壹组 开发文档

**本前端网页采用[vue.js](https://cn.vuejs.org/)框架。**

### [Nuxt.js](https://nuxtjs.org/)

作用：服务端预渲染，路由。

### vuex + axios + js-cookie + vuex-persistedstate

作用：作为用户管理系统连接评论和收藏后台，记录JWT的token。

### jquery + swiper

作用：美化页面

### element-ui + ant-ui

作用：UI framework

### Krpano

作用：为全景提供解决方案





**本网页后端采用GO语言。**

### Gin

作用：轻量级web框架。



### Gorm + MySQL

作用：ORM与数据库。



### Gin-Session + jwt-session

作用：用户验证。
